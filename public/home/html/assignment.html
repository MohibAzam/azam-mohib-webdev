WAM

A simple Wordpress-like website for creating websites without extensive coding

Link to the Assignment Link to the Github Repository

To aid us in understanding Web Development, our homework assignments revolved around incrementally working on a Web Application that would allow users to build their own websites without necessarily needing to code. I felt this was a rather solid way to help familiarize myself with the various concepts presented in class, and the context of a web application made the relationships between objects, as well as the necessary Services, APIs, and Models, easy to spot and understand. I found myself frequently referring back to this work when trying to develop my final project.

I felt my resulting application worked rather solidly. It could performmost of the tasks that were designated by the assignment, be it setting up websites, the pages in the website, and the "widgets" of each page (that is, the replacements for traditional HTML elements). I also found it very easy to implement security in the website through encrypting passwords and session tracking. Of course, there was also the fact that this was implemented as a single-page application, allowing for a tight strcuture in my design with the HTML views, their controllers, the models housing all of the data, and the Client- and Server-side services to connect them all together.

In terms of improvements I could've made, I felt there may have been some issues when it came to managing relationships among objects. In addition, I had some issues in fully implemented the JQuery elements needed to complete the assignments and, as a result of my clear unfamiliarity and a lack of time to fix that, this unfortunately resulted in me avoiding the use of JQuery for my final project. In all other respects, however, I felt my implementation was largely a success, and it certainly helped pave the way to my final project.
